# NYC Listings

Personal knowledge base for NYC apartment hunting.

- `listings/` — one markdown file per listing with structured frontmatter (address, coords, price, beds, contact, etc.)
- `listings/INDEX.md` — comparison table across all listings, sorted by distance to NYU
- `neighborhoods/` — per-neighborhood notes
- `Neighborhoods.md` — neighborhood overview
- `Schedule-vs-Housing.md` — commute / schedule analysis
- `CLAUDE.md` — instructions for Claude on how to add and maintain listings

## Quick start

Open this folder in Claude Code and send a screenshot of a listing. Claude will extract the data, file it under `listings/` with consistent frontmatter, and update `INDEX.md`.
